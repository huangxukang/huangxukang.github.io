# Sass高级语法

# @at-root跳出嵌套

Sass3.3.0中新增的功能，用来跳出选择器嵌套，默认所有的嵌套，继承所有上级选择器，但有了这个就可以跳出所有上级选择器

.parent-2 {
  color:#f00;
  @at-root .child {
    width:200px;
  }
}

上面的代码经过编译后，会生成如下代码

.parent-2 {
	color:#f00;
}
.child{
    width:200px;
}

默认情况下@ar-root不能跳出@media的嵌套，如果要跳出这些嵌套，需要使用@at-root (without: media)

@media only screen and (max-width:768px) {
  .parent2{
    color:#f00;
    @at-root (without: media) {
      .child2 {
        width:200px;
      } 
    }
  }
}

上面的代码经过编译后，可以生成如下代码

@media only screen and (max-width: 768px) {
  .parent2 {
    	color: #f00; 
    } 
}
.child2 {
    width: 200px; 
}

# @each list遍历

@each $var in

其中 $var表示变量，而list和map表示list类型数据和map类型数据，Sass 3.3.0新加入了多字段循环和map数据循环。

$imgList: "01", "02", "03", "04";
@each $index in $imgList{
    .icon-#{$index}{
        background-image: url(images/#{$index}.jpg);
    }
}

上面的代码经过编译后，可以生成如下代码

.icon-01 {
  background-image: url(images/01.jpg); 
}
.icon-02 {
  background-image: url(images/02.jpg); 
}

.icon-03 {
  background-image: url(images/03.jpg); 
}
.icon-04 {
  background-image: url(images/04.jpg); 
}

上面的@each在循环时，使用的是普通list，也可以使用map健值对来进行遍历

$imgList:(1:"img1",2:"img2",3:"img3",4:"img4");
@each $index,$content  in $imgList{
    .div#{$index}{
        background-image: url(images/#{$content}.jpg);
    }
}

上面的代码经过编译后，可以生成如下代码

.div1 {
  background-image: url(images/img1.jpg); 
}
.div2 {
  background-image: url(images/img2.jpg); 
}
.div3 {
  background-image: url(images/img3.jpg);
}
.div4 {
  background-image: url(images/img4.jpg); 
}

# @content占位

@content占位符一般情况下是指在混合器当中，写混合器内部样式时，不确定内部样式如何去操作，这时，就会使用混合器中的占位@content来完成

@mixin flex-box($dir){
    display: flex;
    flex-direction: $dir;
    @content;
}
.div1
{
    @include flex-box(row){
        justify-content: center;
        align-items: center;
    }
}

上面的代码经过编译后，可以生成如下代码

.div1 {
  display: flex;
  flex-direction: row;
  justify-content: center;
  align-items: center; 
}

# 函数

Sass定义了很多函数可供使用，当然也可以自己定义函数

$baseFontSize:      10px !default;
$gray:              #ccc !defualt; 
@function pxToRem($px) {
  @return $px / $baseFontSize * 1rem;
}
body{
  font-size:$baseFontSize;
  color:lighten($gray,10%);
}
.test{
  font-size:pxToRem(16px);
  color:darken($gray,10%);  //注意这里的darken是系统的方法
}

上面的代码经过编译后，可以生成如下代码

body{
  font-size:10px;
  color:#E6E6E6;
}
.test{
  font-size:1.6rem;
  color:#B3B3B3;
}

# 运算

Sass具有运算的特性，可以对数值型的Value(如：数字、颜色、变量等)进行加减乘除四则运算

注意运算符前后请留一个空格，不然会出错

$baseFontSize:          14px !default;
$baseLineHeight:        1.5 !default;
$baseGap:               $baseFontSize * $baseLineHeight !default;
$halfBaseGap:           $baseGap / 2  !default;
$samllFontSize:         $baseFontSize - 2px  !default;

# 条件判断

@if判断

@if可一个条件单独使用，也可以和 @else结合多条件使用

$type:1;
p {
  @if $type == 1 {
    color: blue;
  } @else if $type == 2 {
    color: red;
  } @else if $type == 3 {
    color: green;
  } @else {
    color: black;
  }
}

上面的代码经过编译后，可以生成如下代码

p{
    color:blue;
}

# for循环

for循环有两种形式，分别为： @for $var from through和 @for $var from to

$i表示变量，start表示起始值，end表示结束值，这两个的区别是关键字through表示包括end这个数，而to则不包括end这个数

@for $i from 1 through 3 {
  .div#{$i} { width: 2em * $i; }
}

上面的代码经过编译后，可以生成如下代码

.div1 {
  width: 2em; 
}
.div2 {
  width: 4em; 
}
.div3 {
  width: 6em; 
}

# 三目运算符

if($condition, $if_true, $if_false)

三个参数分别表示：条件，条件为真的值，条件为假的值

$type:1;
p{
    color:if($type==1,red, black);
}

上面的代码经过编译后，可以生成如下代码

p {
  color: red; 
}
