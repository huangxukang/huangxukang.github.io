# CSS过渡

过渡指一个元素由一个状态转变一另一个状态过程，主要指的是Css里面的一个属性transition

# 过渡的基本属性

  1. transition-property指的是过渡的属性，它面如果接了多个属性可以用空格来隔开

  2. transition-duration指的是过渡持续的时间

  3. transition-timing-function指属性在变化的时候的过程，它常用属性如下

    • linear这个代表匀速直线

    • ease属性值

    • ease-in属性值

    • ease-out属性值

    • ease-in-out属性值

    除了通过刚刚几种方式设置它的属性值以外，我们还可以直接在页面上面拖出它的效果，得到属性值

  4. transition-delay指的是过渡延时，它可以在这里让过渡的变化等待多长时间

上面的四个属性是过渡的基本属性，在使用的过程当中，可以把它连起来一起写，这样方便快捷一些

.div1{
    transition: width 0.5s linear 5s; 
}

<style type="text/css">
    .div1{
        width: 100px;
        height: 100px;
        background-color: red;
        border: 1px solid black;
        /*需要过渡的属性*/
        transition-property: width;
        transition-duration: 0.5s;
        /*linear匀速直线*/
        transition-timing-function: linear;
        transition-delay: 5s;
    }
    .div1:hover{
        width: 200px;
    }
    /*
    当我们把鼠标放到 div1上面的时候，元素由一个状态变成了另一个状态
    但是这个状态 的切换是一瞬间就完成的，我们希望有一个变化的过程
    * */
    .div2{
        width: 100px;
        height: 100px;
        background-color: blue;
        border: 1px solid black;
        /*
        先出现的时间是过渡时间，后出现的时间是延时等待时间
        * */
        transition: width 0.5s linear 5s; 
    }
    .div2:hover{
        width: 200px;
    }
</style>
<body>
    <div class="div1">
    </div>
    <div class="div2">
    </div>
</body>

# 多个属性的过渡

<style type="text/css">
    .div1{
        width: 100px;
        height: 100px;
        border: 1px solid black;
        background-color: red;
        /*我现在希望宽度与高度都实现过渡 怎么办*/
        transition-property: width height;
        /*我们可以在在这个属性后面接多个属性*/
        transition-duration: 1s;
        transition-timing-function: linear;
        transition-delay: 1s ;
    }
    .div1:hover{
        width: 200px;
        height: 300px;
    }
</style>

如果要执行多属性的过渡，可以使用transition-propery:属性1 属性2

如果所有的属性都要执行过渡，则使用all这个属性值 transition-propery:all

# 过渡的兼容性处理

transition这个属性它是一个Css3的属性，它在不同的浏览器下面都有不同的兼容性

W3C为了保证各个浏览器的兼容性，在使用Css3的时候，需要加上一个特殊的前缀

  1. 以谷歌为核心的浏览器-webkit-例-webkit-transition

    还包含国内360，qq，搜狗,世界之窗

    更包含android平台内置的浏览器

  2. 以IE为标准的浏览器-ms-例-ms-transition

  3. 以火狐为标准的浏览器-moz-例-moz-transition

  4. 以苹果公司平台的safari为核心的浏览器-webkit-例-webkit-transition

    它指苹果的PC端浏览器，也指IOS的手机浏览器

  5. 以欧朋为核心的浏览器-o-例-o-transition

一般情况下，是不加前缀的，因为开发时是以谷歌浏览器在测试，而谷歌浏览器是完全遵守W3C的标准的

# 不能执行过渡的属性

transition可以执行80%以上的属性过渡，但是仍然会有一些属性无法执行过渡，现在列举一些

  1. display:block/none不能执行过渡
  2. z-index不能执行过渡
  3. float不能执行过渡
  4. visibility不能执行过渡
