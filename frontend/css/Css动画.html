# Css动画

Css当中的动画与过渡+变幻非常相像，但是仍然有一些不同点，首先过渡+变幻虽然可以实现某一个元素的动画效果，但是它仍然存在以下几个缺点

  1. 过渡的效果必须要手动的去触发，不能自动执行
  2. 过渡的效果一次触发只能执行一次过渡，不能够重复或多次的执行

正因为存在上面的问题，所以就出现了动画这一种技术 ，它可以不需要手动触发而进行自动执行，可以重复的执行多次如果要使用一个动画效果，那么必须要先定义这一个动画

# 动画定义

在Css里面，动画的定义使用@keyframes来完成，具体语法格式如下

@keyframes 动画名{
    form{
        /*CSS代码 */
        -webkit-transform: rotateZ(0deg);
    }
    to{
        /*CSS代码 */
        -webkit-transform: rotateZ(360deg);
    }
}

from代表动画开始的时候，to代表动画结束的时候

动画在定义的时候除了通过from与to来表示动画的开始与结束状态以外，还可以通过百分比来表示动画进度

@keyframes 动画名{
    @keyframes a{
        0%{
            transform: rotateZ(0deg);
        }
        20%{
            transform: rotateZ(300deg);
        }
        100%{
            transform: rotateZ(360deg);
        }
    }
}

还可以像下面这种方式来进行

@keyframes a{
    0%,20%{
        transform: rotateZ(0deg);
    }
    /*20%{
    transform: rotateZ(0deg);
    }*/
    21%,40%{
        transform: rotateZ(45deg);
    }
    /*40%{
    transform: rotateZ(45deg);
    }*/
    100%{
        transform: rotateZ(360deg);
    }
}

这就说明动画在0%和20%的时候它的状态是一样的

# 动画属性

一个动画定义好了以后就需要通过调用来实现动画效果，动画的调用主要是通过下面几个属性来完成的

  1. animation-name代表动画的名称
  2. animation-duration代表动画执行一次所需要的时间
  3. animation-iteration-count代表动画重复的次数，后面接一个数字代表执行次数，如果需要运行一直执行则需要填写infinite这个值，这个值代表无穷大，默认执行一次【默认值】
  4. animation-timing-function代表动画执行的效果，linear代表匀速，ease【默认值】
  5. animation-delay代表动画的延时执行时间【默认值】为0
  6. animation-play-state代表动画的播放状态
    • running代表运行状态【默认值】
    • paused代表暂停状态
  7. animation-fill-mode代表动画结束时保留在什么状态
    • forwards代表结束状态
    • backwards 代表回到开始状态【默认值】
  8. animation-direction代表动画播放的方向
    • reverse代表反向运行
    • normal代表正常运行【默认值】
    • alternate代表正反方向交替运行
    • alternate-reverse代表反向与正向交替运行

上面的8个属性名还可以简写成一个属性名animation

div{
    animation:动画名 执行时间 【linear】 【等待时间】  【执行次数】 【播放状态】 【播放方向】 【结束状态】
}
  1. 在上面的属性值里面，第一个时间代表执行时间，第二个时间值代表等待时间
  2. 第一个属性值要设置成动画的名称
  3. 后面的属性值需要就添加，不需要可以省略
  4. 后面的属性值没有顺序限制，谁先谁后都可以（时间值除外）

# 动画案例

轮播图案例

<!DOCTYPE html>
<html>
	<head>
		<meta charset="UTF-8">
		<title>动画案例</title>
		<style type="text/css">
			.box{
				width: 639px;
				height: 426px;
				border: 5px solid blue;
				overflow: hidden;
			}
			.clearfix::after{
				content: "";
				display: block;
				clear: both;
			}
			.imgBox{
				height: 426px;
				border: 1px solid red;
				width: 3195px;
				animation: a1 10s infinite;
			}
			.imgBox>img{
				float: left;
			}
			@keyframes a1{
				0%,15%{
					transform: translateX(0px);
				}
				20%,35%{
					transform: translateX(-639px);
				}
				40%,55%{
					transform: translateX(-1278px);
				}
				60%,75%{
					transform: translateX(-1917px);
				}
				80%,100%{
					transform: translateX(-2556px);
				}
			}
			.box:hover>.imgBox{
				animation-play-state: paused;
			}
		</style>
	</head>
	<body>
		<div class="box">
			<div class="imgBox clearfix">
				<img src="img/item1.jpg" alt="" />
				<img src="img/item2.jpg" alt="" />
				<img src="img/item3.jpg" alt="" />
				<img src="img/item4.jpg" alt="" />
				<img src="img/item1.jpg" alt="" />
			</div>
		</div>
	</body>
</html>
