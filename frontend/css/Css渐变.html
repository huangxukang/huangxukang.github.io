# Css渐变

Css里面的渐变指的是背景的渐变，它主要存在三种形式，一种是线性渐变，一种是径向渐变，还有一种圆锥渐变

# 线性渐变

# 基本线性渐变

沿着某一条直线进行颜色的改变，使用的Css属性为linear-gradient

.div1{
    /*
    标准定法
    */
    background:linear-gradient(to 方向,颜色一,颜色二.......);
    /*兼容写法*/
    background: -webkit-linear-gradient(right,颜色二,颜色一);
}

在兼容语法里面，有两点需要注意

  1. 添加各种浏览器特定的前缀-webkit-，-moz-等
  2. 在方向前面去掉to关键字，颜色反着来

在线性渐变里面，它的第一个参数代表的是方向，这个方向可以指一个方向，也可以指两个关键的组合

如左上角to left top，右上角to right top，左下角to left bottom，右下角to right bottom

后面接的是颜色值

# 设置渐变色的尺寸

设置渐变时，可以在某一个特殊颜色上面设置它的间距，这个间距叫渐变色的颜色尺寸

.div5{
	/*
    在下面的代码里面，我们可以看到蓝色会呈现50%的效果
    50%到70%为黄色
	*/
    background: linear-gradient(to right,blue 50%,yellow 70%,red,orange 100%);
}

注意事项：

在这里，它的尺寸最终不能大于100%，每个颜色使用逗号来进行隔开，然后颜色的后面可以跟百分比的值

线性渐变除了使用某一个方向做为颜色的改变方向，还可以使用角度来进行

# 角度线性线变

.div6{
   /*
    在第一个参数上面，我们已经不再写具体的方法了，转而可以使用一个角度来替代
    */
    background: linear-gradient(0deg,red,blue);
}

0deg代表上面

90deg代表右边

180deg代表下边

270deg代表左边

角度线性渐变就弥补我们之前在基本的线性渐变方向单一的问题，它可以通过角度直接指定某一个方向

# 重复线性渐变

当设置某一个区域渐变颜色的时候，可以重复重复用这些渐变色填充

div{
    width: 260px;
    height: 100px;
    border: 1px solid black;
    background: repeating-linear-gradient(to right,blue 10%,red 20%);
}

# 案例

使用线性渐变，完成一个进度条

<div class="progress">
			
</div>

<style type="text/css">
.progress{
    width: 500px;
    height: 10px;
    border: 1px solid gray;
    border-radius: 10px;
    /* 大多数人采用的进度条都用如下的方式来进行 */
    /* background-image:linear-gradient(to right,red 30%,white 30%); */
    background-image: linear-gradient(to right,red,red);
    background-repeat: no-repeat;
    background-size: 95% 100%;
    /*  下面是一个多色的进度条
    background-image: linear-gradient(to right,red 0%,red 30%,blue 30%,blue 70%,green 70%,green 90%,white 90%);
    */
}
</style>

在上面的案例里面，可以看到一个进度条产生了，这里使用了两种方式 ，各有各的优点

# 径向渐变

径向渐变指提的是由内向外的一种扩散渐变方式，使用的属性名是radial-gradient

径向渐变的语法格式是

div{
  /*径向渐变第一个参数是存在的，但是一般可以省略
    它的第一个参数是ellipse椭圆或circle圆
  */
  background:radial-gradient(可选参数,颜色一,颜色二.....);
}

设置径向渐变的时候，如果这个元素它不是正方形，那么径向渐变默认会成为一个椭圆，如果这个时候还想设置成圆，那么，我们就添加第一个参数为circle

.div1{
    background: radial-gradient(circle,red,blue);
}

案例

<style type="text/css">
    div{
        width: 250px;
        height: 250px;
        border: 1px solid black;
        background: radial-gradient(red,blue);
    }
</style>
<div class="div1"></div>

径向渐变与线性渐变很相似，它也可以设置百分比的值，也可以设置多个颜色值

# 重复的径向渐变

与之前讲过的重复的线性渐变相似

repeating-radial-gradient是重复径向渐变的属性名，设置方式与重复线性渐变一样

.div1{
    width: 300px;
    height: 300px;
    border: 1px solid black;
    background: repeating-radial-gradient(red 10%,blue 20%);
}

题外话

虽然渐变是一个颜色，但是不能通过background-color来设置，背景色的渐变其关把它当成了一个图片，所以，要使用background-image这个属性或background属性

background-image可以设置多个图片，所以下面的代码它是成立的

div{
    /*下面是两个渐变叠加在了一起*/
    background: linear-gradient(to right,rgba(0,0,255,0.5),rgba(255,0,0,0.1)),radial-gradient(deeppink,white);
}

可以统一简写成background

div{
    width: 300px;
    height: 200px;
    border: 1px solid black;
    /*background:red;*/
    background: url(imgb/logo.png) no-repeat right bottom;
}

url代表设置backround-image属性，no-repeat代表设置background-repeat，right，bottom代表设置background-position属性

# conic-gradient圆锥渐变

它的语法格式如下

{
    /* Basic example */
    background: conic-gradient(deeppink, yellowgreen);
}

# 与线性渐变及圆锥渐变的异同

那么它和另外两个渐变的区别在哪里呢？

  • linear-gradient线性渐变的方向是一条直线，可以是任何角度
  • radial-gradient径向渐变是从圆心点以椭圆形状向外扩散 而从方向上来说，圆锥渐变的方向是这样的：

划重点：

从图中可以看到，圆锥渐变的渐变方向和起始点。起始点是图形中心，然后以顺时针方向绕中心实现渐变效果。

# 使用conic-gradient实现颜色表盘

从上面了解了 conic-gradient 最简单的用法，我们使用它实现一个最简单的颜色表盘。

conic-gradient 不仅仅只是从一种颜色渐变到另一种颜色，与另外两个渐变一样，可以实现多颜色的过渡渐变。

由此，想到了彩虹，可以依次列出 赤橙黄绿青蓝紫七种颜色：

  • conic-gradient: (red, orange, yellow, green, teal, blue, purple)

上面表示，在圆锥渐变的过程中，颜色从设定的第一个 red 开始，渐变到 orange ，再到 yellow ，一直到最后设定的 purple 。并且每一个区间是等分的。

再给加上 border-radius: 50% ，假设我们的Css如下：

{
    width: 200px;
    height: 200px;
    border-radius: 50%;
    background: conic-gradient(red, orange, yellow, green, teal, blue, purple);
}

看看效果：

# 配合百分比使用

当然，可以更加具体的指定圆锥渐变每一段的比例，配合百分比，可以很轻松的实现饼图。

假设我们有如下Css：

{
    width: 200px;
    height: 200px;
    background: conic-gradient(deeppink 0, deeppink 30%, yellowgreen 30%, yellowgreen 70%, teal 70%, teal 100%);
    border-radius: 50%;
}

上图，分别指定了 0~30%，30%~70%，70%~100% 三个区间的颜色分别为 deeppink(深红)，yellowgreen(黄绿) 以及 teal(青) ，可以得到如下饼图：

当然，上面只是百分比的第一种写法，还有另一种写法也能实现：

{
    background: conic-gradient(deeppink 0 30%, yellowgreen 0 70%, teal 0 100%);
}

这里表示 ：

  1. 0-30% 的区间使用deeppink
  2. 0-70% 的区间使用yellowgreen
  3. 0-100% 的区间使用teal

而且，先定义的颜色的层叠在后定义的颜色之上。

# 配合 background-size 使用

使用了百分比控制了区间，再配合使用 background-size 就可以实现各种贴图啦。

首先实现一个基础圆锥渐变图形如下：

{
    width: 250px;
    height: 250px;
    margin: 50px auto;
    background: conic-gradient(#000 12.5%, #fff 0 37.5%, #000 0 62.5%, #fff 0 87.5%, #000 0);
}

效果图：

再加上 background-size: 50px 50px;，也就是：

{
    width: 250px;
    height: 250px;
    margin: 50px auto;
    background: conic-gradient(#000 12.5%, #fff 0 37.5%, #000 0 62.5%, #fff 0 87.5%, #000 0);
    background-size: 50px 50px;
}

# 重复圆锥渐变repaeting-conic-gradient

与线性渐变及径向渐变一样，圆锥渐变也是存在重复圆锥渐变 repaet-conic-gradient 的。

假设希望不断重复的片段是 0~30° 的一个片段，它的Css代码是 conic-gradient(deeppink 0 15deg, yellowgreen 0 30deg)

那么，使用了repaeting-conic-gradient之后，会自动填充满整个区域，Css代码如下：

{
    width: 200px;
    height: 200px;
    background: repeating-conic-gradient(deeppink 0 15deg, yellowgreen 0 30deg);
    border: 1px solid #000;
}
